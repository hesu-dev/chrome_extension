# R20-JSONExporter Safari App Scaffold

Standalone Safari Web Extension app scaffold for future Roll20 export on iPhone and iPad.

This source tree is staged into `R20-JSONExporter/release/ios-safari` by the existing deploy pipeline.

ReadingLog integration is intentionally out of scope for this scaffold.

Current scaffold scope:
- Safari popup/content export flow for the loaded Roll20 DOM
- standalone iOS app/extension shell metadata
- shared App Group and native bridge contract placeholders
- App Group inbox preflight and atomic inbox writer source scaffolding
